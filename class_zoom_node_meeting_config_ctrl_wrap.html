◆ SetFloatVideoPos()

static void ZoomNodeMeetingConfigCtrlWrap::SetFloatVideoPos ( const v8::FunctionCallbackInfo< v8::Value > &  args)
static

Set the position of the floating video window when sharing.

Parameters
1.For Windows: hSelfWnd(string) Specifies the window handle of the window itself. For Mac: height(string) Specifies the height of the window.
2.hParent(string) Specifies the window handle of the parent window. If the string is empty, the position coordinate is the monitor coordinate. For Mac: width(string) Specifies the width of the window.
3.left(string) Specifies the X-axis coordinate of the top-left corner of the window
4.top(string) Specifies the Y-axis coordinate of the top-left of the window.
Returns
If the function succeeds, the return value is ZNSDKERR_SUCCESS. Otherwise failed. To get extended error information, see ZNSDKError enum.
Remarks
The value shall be set before the sharing begins. If you set the value during the process of share, the function will not be valid until the next share.
